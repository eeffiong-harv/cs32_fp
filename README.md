# cs32 final project
## _**Tracking Similar Language Use Across Competing News Publications**_

### Final Project Proposal:
> **Q**: How similar is the language used in News Publications' Telegram Posts? (Signaling buzzwords)
> **Q**: Does the use of similar language with competing publications affect viewership?

*Anticipated Steps*:
- Pull posts from telegram channels (according to specified dates)
- Clean posts (stem/lemmanize, organize into centralized table/dict)
- Rank words (tokens) by frequency for each publication
- Identify similar high-ranking words between publications.
- Track how word frequency changes over time (does usage diverge or converge?)
- Compare rating to views.

I expect my final project to output csv files that can be easily transformed into data visualizations and summary reports (exportable as .txt or .md) answering the project questions over a user specified time period.

I plan to begin with completing the scraping script.
